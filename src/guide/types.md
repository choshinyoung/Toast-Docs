# 타입

토스트에서 사용할수 있는 타입들입니다.

## Number

정수와 실수를 모두 포함한 숫자 타입입니다.

### Example

```js
> 1
1
> 1.2
1.2
```

## Text

쌍따옴표(`"`) 또는 따옴표(`'`)로 둘러쌓여진 텍스트 타입입니다.

### Example

```js
> "Hello, World!"
Hello, World!
```


## Boolean

참 또는 거짓을 나타내는 불리언 타입입니다.

### Example

```js
> true
True
> false
False
```

## List

대괄호(`[ ]`)로 둘러싸인 쉼표(`,`)로 구분되는 리스트 타입입니다.

### Example

```js
> [1, 2, 3]
System.Object[]
> ['Hello', 'World']
System.Object[]
```
