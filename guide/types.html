# 타입

토스트에서 사용할수 있는 타입들입니다.

# number

정수를 표현할 수 있는 숫자 타입입니다.

# Example

> 1
1
> -123
-123

# float

실수를 표현할 수 있는 숫자 타입입니다.

# Example

> 3.14
3.14
> -0.01
-0.01

# text

쌍따옴표(") 또는 따옴표(')로 둘러쌓여진 텍스트 타입입니다.

{, }를 사용해 문자열 포매팅을 사용할 수 있습니다.

# Example

> "Hello, World!"
Hello, World!
> 'Toast'
Toast
> "Hello, {1}"
Hello, 1

# boolean

참 또는 거짓을 나타내는 불리언 타입입니다.

# Example

> true
true
> false
false

# List

대괄호([ ])로 둘러싸이고 쉼표(,)로 구분되는 리스트 타입입니다.

# Example

> [1, 2, 3]
[1, 2, 3]
> ['Hello', 'World', 1, [1, 2, 3]]
[Hello, World, 1, [1, 2, 3]]

# function

함수입니다. 코드 여러 줄을 묶을 수 있습니다. 줄은 세미콜론으로 구분됩니다.

# Example

> () {
  print "Wa";
  print "sans"
}
function () { }

# null

비어있는 값을 나타냅니다

# Example

> null
